Running the Demo - C#
=====================

You can run

  demo.bat https://secure.echosign.com/services/EchoSignDocumentService18 <yourApiKey> test

Run 'demo.bat' with no arguments to see the other commands.


Editing/Building the Sample Code
================================

If you want to modify the source, you will need to have at least Microsoft
Visual C# Express installed. You can open the project and build it to produce
an executable. The file you would need to change is App.cs.

Refer to secure.echosign.com.cs when you want data structure constructors
like DocumentCreationInfo(string[] tos, string name, ...).
